# Sanity Studio

## Setup

1. Copy `.env.example` to `.env`.
2. Fill `SANITY_STUDIO_PROJECT_ID` and `SANITY_STUDIO_DATASET`.
3. Install dependencies with `npm install`.
4. Run Studio with `npm run dev`.

## Notes

- Content type configured: `project`
- Frontend expects fields: `title`, `slug`, `summary`, `thumbnail`, `techStack`, `status`, `repoUrl`, `demoUrl`, `documentation`
